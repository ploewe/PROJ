Citation
-------------------------------------------------------------------------------

How to cite PROJ? 
^^^^^^^^^^^^^^^^^^^^^

If you need to cite PROJ in your work or for an assignment, please use the citation style that would be most helpful.

Here are some choices to cite the overall PROJ software project or a specific PROJ version being used used to allow reproducibility:

Cite PROJ as an evolving software project, independent of a specific software release
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. image:: https://zenodo.org/badge/DOI/10.5281/zenodo.5884394.svg
  :target: https://doi.org/10.5281/zenodo.5884394

Use the `overarching Digital Object Identifier (concept DOI: 10.5281/zenodo.5884394) <https://doi.org/10.5281/zenodo.5884394>`__ when

* referring to the PROJ software in general (e.g., in introductions or background sections)
* describing workflows where the exact version is not critical
* citing PROJ as a long-term research tool or infrastructure

The concept DOI will always resolve to the latest version of PROJ and provide a stable reference to the software project over time.





Cite a specific PROJ software release 
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Each PROJ release since release 8.2.1 (2022) has its own specific version DOI identifying a precise, archived snapshot of the software (e.g., a specific software release). All version DOI are linked to the concept DOI of the PROJ project.


**PROJ Release 9.8.1**


.. image:: https://zenodo.org/badge/DOI/10.5281/zenodo.19497389.svg
  :target: https://doi.org/10.5281/zenodo.19497389



Please refer to the  `PROJ Zenodo Landing Page <https://zenodo.org/search?q=parent.id%3A5884394&f=allversions%3Atrue&l=list&p=1&s=10&sort=version>`__ for a list of all specific version DOI for PROJ.

Citation-style Support
^^^^^^^^^^^^^^^^^^^^^^

PROJ references in various citation styles such as APA, Harvard, MLA, Vancouver, Chicago or IEEE are available from the `PROJ Zenodo Landing Page <https://zenodo.org/records/19497389>`__  ("Citation").

**Example: APA Style**

``Evenden, G. I., Rouault, E., Warmerdam, F., Evers, K., Knudsen, T., Butler, H., Taves, M. W., Schwehr, K., Sales de Andrade, E., Karney, C., Couwenberg, S., Dawson, N., Snow, A. D., & Jimenez Shaw, J. (2026). PROJ (9.8.1). Zenodo. https://doi.org/10.5281/zenodo.19497389``

**Example: IEEE Style**

``[1]G. I. Evenden, “PROJ”. Zenodo, Apr. 10, 2026. doi: 10.5281/zenodo.19497389.``

PROJ in Reference Management Systems (e.g. Zotero, Citavi, Endnote, etc.)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
To manage PROJ references with tools like Zotero, Citavi or Endnote, the `PROJ Zenodo Landing Page <https://zenodo.org/records/19497389>`__ offers an export service to several export formats such as BibTex, JSON, DCAT, Dublin Core and more ("Export").
